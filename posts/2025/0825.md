# 使用 LeafLow 部署openList

## 使用链接介绍

openList官网链接：[OpenList - OpenList Docs](https://doc.oplist.org/)

leaflow网站链接：https://leaflow.net/

## 搭建

### docker安装链接

前往openList官网链接，获取docker安装链接：[Use Docker - OpenList Docs](https://doc.oplist.org/guide/installation/docker)

```
docker run --user $(id -u):$(id -g) -d --restart=unless-stopped -v /etc/openlist:/opt/openlist/data -p 5244:5244 -e UMASK=022 --name="openlist" openlistteam/openlist:latest
```

### 导入leaflow

打开leaflow控制台，先选择左侧 1.**部署清单** —> 2.导入docker

![0825-1](https://buc.civer.cn/posts/0825-1.png)

### 转换链接

将上面的docker run命令复制到文本框中，然后点击**转换**

![0825-2](https://buc.civer.cn/posts/0825-2.png)、

### 生成应用

将红框选中部分修改为`openlistteam/openlist:latest`，然后点击**应用/更新**。

![0825-3](https://buc.civer.cn/posts/0825-3.png)

或者直接复制下面内容

```html

```

### 查看运行

选中**容器组**，可以看到容器的运行状态

![0825-4](https://buc.civer.cn/posts/0825-4.png)

### 查看服务

选中**服务管理**，可以看到容器对外开放的容器服务（指集群内）

![0825-5](https://buc.civer.cn/posts/0825-5.png)

### 查看网站

选中**网站管理**，选中**创建网站**

![0825-6](https://buc.civer.cn/posts/0825-6.png)

### 编辑网站信息

**名称**：随便取，如`openlist`

**域名**：填写自己拥有的域名，如`openlist.example.com`

**启用HTTPS**：推荐启用

**选择证书**：默认即可，有自己的域名ssl证书可以自行上传

**后端服务**：选择前面搭建的`openlist`

**服务端口**：选择前面**服务管理**中对应容器`openlist`开放的对应端口即可

最后点击创建网站，即可完成**leaflow平台**侧的配置
